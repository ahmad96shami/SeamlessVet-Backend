# Vendored fonts — attribution

These TrueType fonts are bundled (embedded into the API assembly) so the QuestPDF report exporter
renders Arabic text correctly in any environment (CI, Docker) without relying on system-installed
fonts (M12 task 13 — "PDF export with RTL Arabic support").

- **NotoSansArabic-Regular.ttf**, **NotoSansArabic-Bold.ttf**
  - Family: `Noto Sans Arabic`

software). The full license text is available at the link above and from the upstream font project.
